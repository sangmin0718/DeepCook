# DeepCook

This project provides a simple Android example application written in **Java** with Gradle scripts in **Groovy**. It scans a receipt image, extracts text using Google ML Kit, detects ingredients, and suggests recipes from several cuisines.

## Setup
1. Open the project with Android Studio.
2. Ensure you have the latest Android SDK and build tools installed.
3. Build and run the `app` module on an emulator or physical device.

The application allows the user to pick a receipt image from the gallery. After text recognition, it recommends recipes from the following categories: Korean, Western, Japanese, Chinese, and Late-night snacks.
